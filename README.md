# Netflix Analysis (Python + Tableau Dashboard)
## Project Overview

This project focuses on analyzing Netflix Movies and TV Shows data to extract meaningful insights using Python for data analysis and Tableau for interactive dashboard visualization.

The goal of this project is to understand:

* Distribution of Movies and TV Shows on Netflix
* Popular genres, ratings, and content types
* Country-wise content production
* Trends in content release over the years
* Key contributors such as directors and cast

This project demonstrates data analysis, visualization, and storytelling skills.

## Tools & Technologies Used
* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Tableau

* Tableau Public
* Interactive Dashboards
* Filters & Parameters
* Maps, Bar Charts, Line Charts, Pie Charts

## Dashboard Features

* Movies vs TV Shows Distribution
* Genre-wise Analysis
* Country-wise Content Analysis
* Ratings Distribution
* Year-wise Trends

## Filters & Interactivity

* Content Type Filter (Movie / TV Show)
* Country Filter
* Release Year Filter
* Genre Filter
* Dynamic Dashboard Controls
